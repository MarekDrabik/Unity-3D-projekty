V tomto repository nájdete spúšťacie súbory na dve "hry", ktoré som vytvoril v Unity3D.

	* Mechanická ruka
	* Výstrely šípov

Ovládanie pre mechanickú ruku je takéto:
>**w, s, d, a** = vodorovný pohyb
>**r, f** = zvislý pohyb
>**q, e** = rotácia ruky
>**tlačidlá myši** = otváranie a zatváranie ruky
